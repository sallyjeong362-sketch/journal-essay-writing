# journal-essay-writing

✏️ **나의 글쓰기 노트** — 학생을 위한 저널·에세이 라이팅 앱

주제 선택부터 마인드맵, 서론(Introduction)·본론(Body)·결론(Conclusion) 작성까지
단계별로 안내하는 글쓰기 웹앱입니다. 별도 설치나 빌드 없이 `index.html` 하나로 동작하며,
작성한 글은 브라우저의 localStorage에 저장됩니다.

## 주요 기능

- **주제 & 예시글**: My Favorite Color, My Favorite Sport 등 미리 준비된 주제를 클릭하면
  그 주제에 맞는 예시글과 질문이 자동으로 준비됩니다. 직접 주제를 입력할 수도 있습니다.
- **마인드맵**: 가운데 동그라미에 주제를 적고, 생각나는 아이디어를 동그라미로 확장하며
  브레인스토밍할 수 있습니다. "본론으로 보내기" 버튼으로 마인드맵 내용을 바로 본론 작성에
  활용할 수 있습니다.
- **서론(Introduction)**: Hook/Question → Transition → Topic Sentence 순서로,
  단계별 예시 문장을 참고하며 서론을 완성합니다.
- **본론(Body)**: 문단마다 Main Point → Detail 1 → Detail 2 구조로 작성하고,
  전환어(Transition Words) 도우미로 문장을 자연스럽게 연결합니다. 문단은 자유롭게
  추가·삭제할 수 있습니다.
- **결론(Conclusion)**: 서론의 핵심 생각을 다시 정리(Retell)하고, 주제별 마무리 질문으로
  생각을 확장합니다.
- **완성된 글**: 들여쓰기·문단 간격 등 글쓰기 체크리스트와 문장 수 목표를 확인하고,
  완성된 글을 저장·복사·텍스트 파일 내보내기·인쇄할 수 있습니다.
- **내 글 목록**: 저장한 글을 다시 열어보거나 삭제할 수 있습니다.
- **키즈 / 청소년 모드**: 화면 오른쪽 위 버튼으로 언제든 전환할 수 있습니다.
  - 🧒 **키즈 모드**: 알록달록한 색상과 큰 이모지, 동글동글한 마인드맵 버블 등
    흥미를 자극하는 디자인
  - 🧑‍🎓 **청소년 모드**: 절제된 색상, 넓은 여백, 세리프 글꼴의 완성된 글 미리보기 등
    가독성 중심의 차분한 디자인

## 사용 방법

`index.html` 파일을 브라우저로 열기만 하면 바로 사용할 수 있습니다 (별도 서버나 빌드 과정 불필요).
